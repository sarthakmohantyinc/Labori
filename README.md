# Labori
Destitutus ventis, remos adhibe. Resource Bank's sentient robot helper.

## How to Run
- Add the following variables in the `.env` file
    - `SLACK_BOT_TOKEN`
    - `SLACK_SIGNING_SECRET`
    - `SLACK_TOKEN`
    - `AWS_ACCESS_KEY`
    - `AWS_SECRET_ACCESS_KEY`
    - `AIRTABLE_API_KEY`
- run `npm install`
- run `npm start`
- Slack Bot should be up!
